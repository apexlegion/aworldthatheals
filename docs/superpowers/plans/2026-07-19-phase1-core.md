# A World That Heals — Phase 1 Core Implementation Plan

> **For agentic workers:** Steps use checkbox (`- [ ]`) syntax for tracking. This is a static PWA with no test framework; verification is manual in-browser per the spec.

**Goal:** Build a free, privacy-first, static-PWA core for A World That Heals: intent-driven search, ~18 curated encyclopedia topics, and always-present emergency support.

**Architecture:** Plain HTML/CSS/vanilla JS, no build step. Content lives in per-topic JSON files loaded at runtime. Client-side phrase-matching search. Service worker caches app shell + visited topics for offline. No backend, no accounts, no tracking.

**Tech Stack:** HTML5, modern CSS (custom properties, grid/flex), vanilla ES modules, PWA (manifest + service worker), LocalStorage for dismissible-banner state only.

## Global Constraints

- 100% free, no ads, no affiliate, no accounts, no tracking, no geo-IP.
- Every intervention/resource carries an evidence-status label: `evidence-based` | `emerging` | `traditional`.
- Educational/navigational only; never diagnostic. Crisis resources surfaced prominently and consistently on every page.
- Palette: near-black + deep olive primary surfaces, warm cream/beige secondary panels, single orange-red accent for CTAs/emphasis. Dark default + cream light theme. Mobile-first, WCAG 2.2 AA basics (contrast, keyboard nav, ARIA labels, focus states).
- No inline external dependencies / CDNs — fully self-contained so it works offline and deploys as static files.

## File Structure

```
world-that-heals/
  index.html            # Home: search + topic grid
  topic.html            # Topic detail (reads ?id= param)
  emergency.html        # Emergency support + country selector
  about.html            # Mission, principles, disclaimer
  css/styles.css        # Full design system, dark+light
  js/app.js             # Shared: theme, banner, header, emergency strip
  js/search.js          # Phrase-matching engine
  js/topic.js           # Topic page renderer
  js/emergency.js       # Country selector renderer
  js/data.js            # Fetch helpers + topic index loader
  data/topics/*.json    # 18 topic files
  data/topics-index.json# id/title/blurb/evidenceStatus for grid+search
  data/synonyms.json    # shared synonym expansion for search
  data/crisis-terms.json# crisis-language wordlist
  data/emergency.json   # per-country crisis resources
  manifest.webmanifest
  sw.js                 # service worker
  icons/                # PWA icons
```
